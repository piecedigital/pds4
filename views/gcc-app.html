Guide Cyberclub Chat App
Description

This project was created for a small organization by the name of Guide Magazine. They hold weekly chat sessions online.

I offered my skills to help them out a bit by creating a chat application to replace their current Java applet. The app needed to carry over and improve all of the necessary parts of their previous app, and also add some nifty new ones. The goal was to create a customized, functional, and scalable web app.
