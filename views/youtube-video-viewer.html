YouTube Video Viewer
Description

This project was my first time diving into AngularJS, harnessing it's tools to use the YouTube API and pull channel data.

In this mini-project you have the option to type in the name of a YouTube channel (provided that they have a URL) and view, and navigate through all available videos on that channel.
