Twitch Exe
Description

Twitch Exe is a desktop application for watching livestreams from Twitch.tv. It's built using Electron.
It provides and asynchronous viewing experience so you don't have to refresh or stop watching a streamer just to see who's online! It supports multi-stream viewing and it also supports multitwitch.tv links!
