Twitch Exe is a desktop application for watching livestreams from <a href="http://twitch.tv">Twitch.tv</a>. It's built using <a href="http://electron.atom.io">Electron</a>.<br>It provides and asynchronous viewing experience so you don't have to refresh or stop watching a streamer just to see who's online! It supports multi-stream viewing and it also supports <a href="http://multitwitch.tv/">multitwitch.tv</a> links!
